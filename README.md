# 카드키 관리 시스템 (Card Key Management System)

이 프로젝트는 구글 스프레드시트를 기반으로 한 현장 판매 등록 및 데이터 매칭 시스템입니다.

## 1. 시스템 구조 (데이터 흐름)
현장 직원이 입력한 **'서비스번호'**를 기준으로 관리자 시트의 원천 데이터(계약/고객/청구번호)를 실시간 매칭하여 최종 엑셀 업로드용 데이터를 생성합니다.

## 2. 주요 기능
- **현장 입력창**: 현장 직원이 모바일이나 PC에서 간편하게 판매 내역을 등록합니다.
- **자동 데이터 매칭**: '마스터데이터' 시트의 서비스번호를 키값으로 사용하여 계약 정보를 자동으로 불러옵니다.
- **업로드용 데이터 생성**: 회사 전산 시스템에 바로 업로드할 수 있는 6개 컬럼 구성의 결과 시트를 생성합니다.

## 3. 사용 방법 (설치 가이드)
1. 구글 스프레드시트를 생성합니다.
2. 시트 이름을 각각 `판매내역`, `마스터데이터`로 변경합니다.
3. `마스터데이터` 시트에 원천 데이터를 붙여넣습니다 (A:계약번호, B:고객번호, C:청구번호, D:서비스번호 순).
4. **[확장 프로그램] > [Apps Script]**를 클릭합니다.
5. `Code.gs` 파일에 제공된 Google Apps Script 코드를 복사해 넣습니다.
6. **[파일] > [새로 만들기] > [HTML]**을 선택하고 이름을 `Index`로 한 뒤 제공된 HTML 코드를 복사해 넣습니다.
7. 저장 후 스프레드시트를 새로고침하면 상단에 **🚀 관리자 시스템** 메뉴가 나타납니다.
